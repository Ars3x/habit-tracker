# Habit Tracker — трекер привычек с уведомлениями и геймификацией

Это проект для самосовершенствования. Пользователь создаёт привычки, получает напоминания (даже при закрытой вкладке), зарабатывает очки и уровни.

## Что уже сделано (этап 2)

### Бэкенд на FastAPI

Мы установили:
- **FastAPI** — современный веб-фреймворк для API на Python. Он автоматически проверяет типы данных, генерирует документацию (Swagger) и работает быстро.
- **Uvicorn** — ASGI-сервер, который запускает FastAPI-приложение. Он поддерживает асинхронность и перезагрузку при изменениях (`--reload`).

### Создано два эндпоинта:

| Метод | Эндпоинт | Что возвращает |
|-------|----------|----------------|
| GET   | `/`      | `{"message": "Привет, трекер привычек!"}` |
| GET   | `/health` | `{"status": "ok"}` |

### Как запустить бэкенд сейчас:

```bash
uvicorn backend.app.main:app --reload
